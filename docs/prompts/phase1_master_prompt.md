# kview Phase 1 --- Master Prompt (Platform Foundation)

## Context

kview is a **local‑first Kubernetes UI client written in Go with a React
frontend**.

Architecture: Go backend → HTTP/WebSocket server → React UI → Browser

Milestone 2 is complete.

Phase 1 introduces platform foundations: - Desktop webview launcher - UI
theming (light / dark / system) - Bottom activity panel - Runtime /
activity backend scaffolding

This phase builds infrastructure only.
